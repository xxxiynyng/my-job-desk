# Pickd 타이포그래피 토큰 마이그레이션 계획

> 목적: 코드 곳곳에 손으로 박힌 폰트 크기(`text-[13px]` 등 약 313곳)를 **디자인 시스템 SSOT 3장이 이미 확정한 위계**에 이름표(토큰)를 붙여 단일 출처화. 크기는 바꾸지 않으므로 **시각 변화 0**(예외 1건: 에디터 제목 16→15px 통일).
> 확정 결정(2026-07-05): ① 토큰 이름 = **역할 기반** ② 제목 = **15px로 통일**.

## 1. 최종 토큰 스케일 (tailwind.config.ts `theme.extend`에 추가)

```ts
fontSize: {
  micro:   ['9px',  { lineHeight: '1'   }], // 캘린더 마이크로 배지
  mini:    ['10px', { lineHeight: '1.2' }], // 카운트 숫자
  chip:    ['11px', { lineHeight: '1.3' }], // 태그·키워드·유형 칩  [SSOT 5-2]
  body:    ['13px', { lineHeight: '1.5' }], // 데이터 행·메타       [SSOT 3장]
  title:   ['15px', { lineHeight: '1.3' }], // 모달·섹션 제목 (h2)
  heading: ['26px', { lineHeight: '1.1' }], // 페이지 H1
},
```

기존 Tailwind 기본값(`text-xs`=12px, `text-sm`=14px, `text-base`=16px)은 그대로 유지된다. 위 6개만 추가하며 전부 현재 렌더 px와 동일하다.

## 2. 치환 매핑표 (현재 → 토큰)

| 현재 값 | 실측 횟수 | 역할 | → 토큰 | 비고 |
|---|---|---|---|---|
| `text-[11px]` | 122 | 칩·태그 | `text-chip` | 신규, 동일 px |
| `text-[12px]` | 58 | 헤더·배지 | `text-xs` | 기존 토큰(공짜) |
| `text-[10px]` | 57 | 카운트 숫자 | `text-mini` | 신규, 동일 px |
| `text-[13px]` | 56 | 데이터 행·메타 | `text-body` | 신규, 동일 px |
| `text-[9px]` | 10 | 캘린더 마이크로 배지 | `text-micro` | 신규, 동일 px |
| `text-[26px]` | 5 | 페이지 H1 | `text-heading` | 신규, 동일 px |
| `text-[15px]` | 3 | 모달·섹션 제목 | `text-title` | 신규, 동일 px |
| `text-[16px]` | 1 | 에디터 제목(DetailEditor) | `text-title` | ⚠️ 16→15px (유일한 시각 변화) |
| `text-[14px]` | 1 | 1차 항목명 | `text-sm` | 기존 토큰(공짜) |
| `bg-[#F8FAFC]` | 5 | 테이블 헤더 행 배경 | `bg-slate-50` | 정확히 일치(SSOT surface-header-row) |

> 실측은 세션 스냅샷 기준. 터미널에서 적용 전 라이브 레포에서 다시 grep해 최신 횟수로 확인할 것(그 사이 파일이 추가됨).

## 3. 적용 순서 (파일이 아니라 토큰 단위 커밋 — 되돌림·리뷰 쉬움)

1. `chore(ds): 타이포 토큰 6종 tailwind.config 정의` — 스케일만 추가(아직 사용처 없음).
2. `refactor(ds): text-[12px]→text-xs, text-[14px]→text-sm` — 기존 토큰으로, 시각 변화 0.
3. `refactor(ds): 칩·본문·카운트·마이크로·H1 토큰 치환` — chip/body/mini/micro/heading, 시각 변화 0.
4. `refactor(ds): 제목 text-title 통일(16→15)` — **유일한 시각 변화**라 단독 커밋. DetailEditor 제목 스크린샷 첨부.
5. `refactor(ds): 헤더 행 배경 bg-[#F8FAFC]→bg-slate-50` — 시각 변화 0.

## 4. 자가검증 (매 커밋)

- `npx tsc -p tsconfig.app.json --noEmit` 신규 에러 0 + `npm run build` + `npm run test`.
- 스크린샷 대조(시각 회귀 확인): ① 탭1/탭2 테이블(칩 11·헤더 12·본문 13·항목명 14) ② 캘린더 배지(9·10) ③ 페이지 H1(26) ④ 모달 제목(15). 4번 커밋은 DetailEditor 제목 before/after 필수.
